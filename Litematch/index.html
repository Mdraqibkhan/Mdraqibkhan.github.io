Abstract

LiteMatch is a lightweight stereo-matching framework designed for strong zero-shot generalization. It combines a two-stream feature encoder, cross-view correspondence learning, cost-volume stabilization, and an efficient disparity refinement module. The model separates semantic and high-frequency evidence, stabilizes matching before refinement, and predicts detailed disparity maps with only 3.36M parameters. Across unseen stereo benchmarks, LiteMatch aims to preserve sharp boundaries and fine structures while remaining substantially smaller than competing methods.

Method Overview

Feature extraction and stabilization. Encoder E1 learns cross-view correspondence, while encoder E2 preserves high-frequency cues using the proposed frequency pathway. Their features construct a cost volume that is regularized with Cross-View Correspondence (CVC) loss.

Disparity refinement. A transformer decoder combines disparity context and correspondence features to predict a base disparity, followed by lightweight iterative refinement for the final output.
